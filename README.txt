JSONify:README

New Attribute: Creates a new attribute field presentet as key and value.

Key: The attribute name of the json object
Value: The value that the object key contains

JSONify: Writes your attribute fields as an json object to a .json file.

Warning: You have to make your .json file before writing to it. If a .json file is
empty, then it will write your object as first entry to the list. If there already
is data in you .json file, it will reproduce the .json data, and then after add your
new object to the list.
